# F29-Retaliator
A modern flight-combat game project inspired by F-29 Retaliator (https://store.steampowered.com/app/1302270/F29_Retaliator/)

The source code is released under Apache License 2.0 as specified in the LICENSE file.

## Some Background
F-29 Retaliator, released in 1989 by Digital Image Design, is remembered as an early and influential flight-combat game from that era.

![F29 Retaliator](/retal.jpg)

This project focuses on recreating the spirit of classic flight-combat gameplay with modern systems and visuals.
